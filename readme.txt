== Installation ==

Installing should be a piece of cake and take fewer than 3 minutes.

1. Install AddFreeStats plugin either via the WordPress plugin directory, or by uploading the files to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on "Settings" link and on AFSWP option.
4. AFS plugin page is displayed. Click on "Get a website ID" link.
5. Enter the website ID into the website ID field.
6. Click on "Update option" button.
